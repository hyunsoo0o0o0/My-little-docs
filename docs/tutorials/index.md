# Tutorials

Step-by-step guides demonstrating how to use SSTDV-Project tools for real-world research problems.

## Available Tutorials

### [Brain Imaging Analysis](tutorials/brain-imaging.html)
Complete pipeline for analyzing brain imaging data using BrainShapeToolKit and DiffAM. Covers preprocessing, shape modeling, and visualization.

### [Breast Cancer Detection](tutorials/breast-cancer.html)
End-to-end workflow for breast cancer detection using synthetic data augmentation with DiffuAug and evaluation with EvalDistSim.

### [Environmental Data Analysis](tutorials/environment.html)
Sea surface temperature modeling and analysis using KIOST-SST-EVL. Includes data ingestion, extreme value analysis, and visualization.

## Prerequisites

Before starting any tutorial, ensure you have:

- Python 3.9+ installed
- Git installed
- Access to the respective dataset (links provided in each tutorial)

## Getting Help

If you encounter issues, please open an issue on the relevant repository:
- [BrainShapeToolKit Issues](https://github.com/SSTDV-Project/BrainShapeToolKit/issues)
- [DiffAM Issues](https://github.com/SSTDV-Project/DiffAM/issues)
- [KIOST-SST-EVL Issues](https://github.com/SSTDV-Project/KIOST-SST-EVL/issues)
